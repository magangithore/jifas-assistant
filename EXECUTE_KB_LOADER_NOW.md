## ?? TROUBLESHOOTING

### Error: SQL Server Connection Failed
```
Fix:
1. Open SQL Server Management Studio
2. Verify (localdb)\MSSQLLocalDB is running
3. Create database JIFAS_Assistant if missing
4. Check appsettings.json connection string
```

### Error: Ollama Not Responding
```
Fix:
1. Start Ollama: ollama serve
2. Verify: curl http://10.0.12.54:11434/api/tags
3. Pull model: ollama pull qwen3-embedding:4b
4. Check IP: ping 10.0.12.54
```

### Error: Project Build Failed
```
Fix:
1. Run from project root directory
2. Ensure .NET 10 SDK installed: dotnet --version
3. Clear cache: dotnet clean
4. Restore packages: dotnet restore
5. Try again
```

### Script Hangs or Takes Too Long
```
Normal behavior:
- Chunking: 10-30 seconds
- Embedding (456 chunks): 15-25 minutes
- Database inserts: 2-5 minutes
- Total: ~20-30 minutes

If > 45 minutes:
1. Check Ollama CPU usage (should be high)
2. Check SQL Server CPU/memory
3. Ctrl+C to stop and retry
```
