## ?? Troubleshooting

### Issue 1: API Not Running
```
Error: Connection refused on localhost:5000
```
**Fix:** `dotnet run` di folder Jifas.Assistant

### Issue 2: Ollama Server Timeout
```
Error: HTTP 504 Gateway Timeout
```
**Fix:** Check Ollama server at 10.0.12.54:11434

### Issue 3: Empty Response
```
Response: ""
```
**Fix:** Check Knowledge Base configuration
