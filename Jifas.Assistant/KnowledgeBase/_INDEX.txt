JIFAS Knowledge Base — Index & Panduan Navigasi
=================================================
File ini adalah indeks dari seluruh konten Knowledge Base JIFAS.
Gunakan untuk menemukan file yang relevan dengan pertanyaan user.

=================================================
OVERVIEW & REFERENSI UMUM
=================================================
Jifas.txt
  -> Deskripsi lengkap JIFAS, semua modul, URL akses, cara login,
     alur approval, roles, aturan penting, kontak eskalasi.
     [BACAAN PERTAMA untuk pertanyaan umum tentang JIFAS]

userguide.txt
  -> Panduan operasional untuk user: cara login/logout, navigasi menu,
     penggunaan modul, troubleshooting umum, tips penggunaan efisien.
     [GUNAKAN untuk pertanyaan "bagaimana cara..." yang sifatnya operasional]

=================================================
MODUL: MASTER DATA
=================================================
Master\General.txt      -> Overview modul master, daftar sub-modul, hak akses
Master\Company.txt      -> Data perusahaan dan cabang
Master\Employee.txt     -> Data karyawan (dipakai di PUM, payroll)
Master\Vendor.txt       -> Data supplier/rekanan bisnis, cara tambah/edit vendor
Master\Division.txt     -> Struktur divisi organisasi
Master\Departement.txt  -> Struktur departemen
Master\COA.txt          -> Chart of Accounts, kode akun, hierarki akun
Master\AccPeriod.txt    -> Buka/tutup periode akuntansi bulanan/tahunan
Master\ListCoa.txt      -> Tampilan daftar COA aktif (view-only)
Master\ReportSetup.txt  -> Mapping COA ke template laporan keuangan
Master\Budget.txt       -> Anggaran tahunan per COA/departemen
Master\Roles.txt        -> Hak akses user (WMTR, USER, USRL, FINA, SPVR, dll)

=================================================
MODUL: INVOICE
=================================================
Invoice\Create.txt            -> Cara membuat invoice baru, field-field, error handling
Invoice\Finance.txt           -> Proses finance checking/approval invoice
Invoice\HeadApproval.txt      -> Proses head/director approval invoice
Invoice\Tax.txt               -> Tax approval dan tax correction di invoice
Invoice\ApprovalIncomplete.txt-> Penanganan invoice yang approval-nya tidak lengkap

=================================================
MODUL: PUM (Perjalanan Uang Muka)
=================================================
Pum\Pengajuan.txt         -> Cara pengajuan PUM baru, field-field, workflow
Pum\HeadApproval.txt      -> Proses approval Head/Supervisor untuk PUM
Pum\TaxApproval.txt       -> Tax approval untuk PUM
Pum\PPUM&Realization.txt  -> Proses realisasi dan settlement PUM

=================================================
MODUL: RECEIVING
=================================================
Receiving\Create.txt                  -> Cara buat dokumen penerimaan baru (RV)
Receiving\ReceiveOfSales.txt          -> Penerimaan dari transaksi penjualan
Receiving\TaxApproval.txt             -> Tax approval untuk receiving (NPWP, alamat WP)
Receiving\ApprovalOfUnidentifiedRv.txt-> Approval RV yang tidak teridentifikasi

=================================================
MODUL: PAYMENT
=================================================
Payment\PaymentInvoice.txt -> Pembayaran invoice vendor/supplier
Payment\PaymentPUM.txt     -> Distribusi/cairkan uang muka PUM
Payment\ListBg.txt         -> Daftar Bank Garansi (BG)

Cashbank\Payment.txt       -> Proses payment di cashbank
Cashbank\PaymentTax.txt    -> Payment dengan aspek perpajakan
Cashbank\Receive.txt       -> Penerimaan di cashbank
Cashbank\ReceiveTax.txt    -> Penerimaan dengan aspek perpajakan

=================================================
MODUL: OVER BUDGET
=================================================
Budget\BudgetStatusReference.txt -> Penjelasan status OK/CM/CA/CY, cara penanganan
OverBudget\FinanceApproval.txt   -> Approval Finance untuk transaksi over budget
OverBudget\HeadApproval.txt      -> Approval Head untuk transaksi over budget

=================================================
MODUL: ACCOUNTING
=================================================
Accounting\JIFAS_ACCOUNTING_COMPLETE_GUIDE.txt
  -> Panduan lengkap modul akuntansi: jurnal harian, jurnal memorial,
     jurnal kas (CAJE/PAJE), penyesuaian, pembatalan, laporan neraca,
     trial balance, troubleshooting accounting, tips penggunaan.

=================================================
MODUL: REPORT (LAPORAN)
=================================================
Report\BudgetCard.txt          -> Laporan kartu anggaran per cost center
Report\BudgetCommited.txt      -> Laporan komitmen anggaran
Report\BudgetPayment.txt       -> Laporan realisasi pembayaran vs anggaran
Report\BudgetRealization.txt   -> Laporan realisasi anggaran
Report\BudgetReceive.txt       -> Laporan penerimaan terkait anggaran
Report\CommitedRealization.txt -> Laporan komitmen vs realisasi aktual
Report\DailyCashflow.txt       -> Laporan arus kas harian
Report\CashbankDetail.txt      -> Detail transaksi kas dan bank
Report\CashbankRecap.txt       -> Rekap kas dan bank per periode
Report\SaldoBukuBank.txt       -> Rekonsiliasi buku perusahaan vs bank
Report\DepositoAktif.txt       -> Daftar deposito yang sedang berjalan
Report\InquiryAp.txt           -> Inquiry saldo hutang ke vendor (AP)
Report\InquiryAr.txt           -> Inquiry saldo piutang dari customer (AR)
Report\InquiryCb.txt           -> Inquiry saldo kas dan bank (CB)
Report\InquiryPUM.txt          -> Inquiry saldo uang muka karyawan
Report\RealisasiPUM.txt        -> Laporan realisasi pengeluaran dinas
